# SearchUsername
SearchUsername is a simple application to see user information with username.

https://searchusername.herokuapp.com/

Requires Ruby 1.9.2 or higher.
  
Does not require Database

```Steps
<br>
Checkout the repository
<br>
bundle install
<br>
rails s

Example:

Searching by "Pushpalata_C"
Will give you twiiter,rubygem and github result for user name "Pushpalata_C"
